Update:
Die in Coop eingegebenen Konfigurationen werde gespeichert in:
CoopClient.app/Contents/MacOS/workspace/.metadata/plugins/org.eclipse.core.runtime/.settings/com.weiglewilczek.coop.client.prefs
Bei einem Update auf eine andere Programmversion, bitte diese Datei (bzw. dern Ordner org.eclipse.core.runtime) manuell in das neue Programm kopieren.

Known Bug:
Vor dem Start muss dem Owner noch Execution-Rechte für die Datei CoopClient.app/Contents/MacOS/CoopClient hinzugefügt werden.
Also:
chmod o+x ....CoopClient.app/Contents/MacOS/CoopClient
